MATLAB

If you have MATALB2017a or more later, you can use MuPAT on MATALAB. Currently, macOS is supported.

GitHub: https://github.com/HotakaYagi/MuPAT (available from end of June 2019)
MATLAB toolbox: Download MuPAT3.zip
UserGuide: User's guide in Japanese or English

Scilab

If you have Scilab, you can use MuPAT on Scilab. Scilab version does not support any parallelization.

Scilab toolbox: https://atoms.scilab.org/toolboxes/DD_QD
