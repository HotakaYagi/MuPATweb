MuPAT

High-precision Arithmetic Environment

  This project helps user to compute fast some matrix and vector operation with high-precision. While rounding errors are unavoidable in floating point arithmetic, they can be reduced through the use of high-precision arithmetic. MuPAT is based on double-double (DD) and quad-double (QD) arithmetics. DD (QD) is the way to represent a quadruple (octuple) precision number with two (four) double precision numbers.

Ease-to-Use (interactive and overloading operators)

  Easy-to-use high-precision arithmetic is also important for end users. In response to this need, our team developed MuPAT, an open-source interactive multiple precision arithmetic toolbox, which runs for use with the MATLAB and Scilab computing environments. You can use same operators (-, +, *, /, ', [,], [;], etc), and comparisons (<, <=, ==, ~=, >=, >) for double-double, quad-double, and mixed-precision floating point operations.

Fast Computation

  Since DD and QD arithmetics require from 10 to 600 double precision floating point operations for each operation, the computation time is often result in excessively long time. In order to accelerate processing, we applied three parallel processing features, FMA, AVX2, and OpenMP for some basic matrix and vector operations. As a result, we can compute DD and QD operations with 16 to 17 times faster than without acceleration.
